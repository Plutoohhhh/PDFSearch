# PDFSearch


## Dependencies

1. Python3 (>=3.10)
2. PySide6




## Usage

Run Script :

   ```shell
   python3 main.py
   ```

Pack App :
    macos
   ```shell
   pyinstaller main.py --clean -D -w -y -i AppIcon.icns -n TDSSearch
   ```
   Windows
  ```shell
   pyinstaller main.py --clean -F -w -y -i AppIcon.icns -n TDSSearch
   ```
